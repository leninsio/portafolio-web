### Institutional Predator Mark IV

The ZTC Trading Bot is a sophisticated automation tool designed for trading Synthetic Indices on the M15 timeframe. It implements a rigorous strategy based on Zone identification, Trend analysis, and Confirmation signals.

**Key Features:**
- **Compute:** Scalable execution on Hugging Face Spaces using Docker containers.
- **Persistence:** Real-time data management via Firebase Firestore for cloud-based state tracking.
- **Async Execution:** Built with Python's `asyncio` and the Deriv API for high-performance, non-blocking operations.
- **Live Notifications:** Integrated Telegram Bot API for real-time trade alerts and performance reporting.
- **Zero-VPS Architecture:** Optimized to run on free-tier cloud services without sacrificing reliability.
