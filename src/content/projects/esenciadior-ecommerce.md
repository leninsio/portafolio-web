### Premium Retail Experience

Esencia Dior is a full-stack e-commerce solution tailored for the luxury market. It provides a seamless shopping experience for high-end fragrances, backed by a powerful administrative suite.

**Technical Stack:**
- **Frontend:** Next.js with TypeScript and Tailwind CSS for a responsive, high-performance UI.
- **Backend:** Firebase ecosystem (Authentication, Firestore, Storage) providing a serverless infrastructure.
- **State Management:** Zustand for efficient client-side state handling.
- **Design:** Modern aesthetics using Lucide React icons and custom animations.
- **Admin Suite:** Comprehensive dashboard for product management, stock control, and order tracking.
