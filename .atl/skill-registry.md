# Skill Registry - portafolio-web

## Compact Rules

- **Astro**: Follow Astro best practices for components and pages. Use `.astro` files for UI.
- **Tailwind CSS**: Use Tailwind CSS for styling. Prefer `@tailwindcss/vite` integration.
- **TypeScript**: Adhere to `astro/tsconfigs/strict`.

## User Skills

| Trigger | Skill | Description |
|---|---|---|
| `astro`, `.astro` | astro-dev | General Astro development rules |
| `tailwind`, `css` | tailwind-styling | Styling conventions |
